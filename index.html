TradingView webhooks require port 80 (HTTP) or port 443 (HTTPS). Use a cloud deploy — NOT localhost.
1. Go to railway.app → New Project → Deploy from GitHub
2. Upload server.js + package.json to a GitHub repo
3. Railway auto-assigns a public URL like https://sui-bot.up.railway.app
4. Add env var: WEBHOOK_SECRET = sui_smc_secret_2024
5. Paste your Railway URL below ↓
